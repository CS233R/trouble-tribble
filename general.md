#General
####Define Variables
```ruby
puts "What is your age?"
age = gets
puts "Your age is #{age}."

#Multiple/Parallel Assignment
a, b, c = 10, 20, 30 #a == 10, b == 20, c == 30

#Parallel Variable Swapping
a, b = b, c #a == b and b == c

```

```ruby
Name file with extension ".rb"

```
